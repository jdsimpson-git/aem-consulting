# Life Insurance "Fast Food" Wizard - Streamlined UI Flow

## Design Philosophy

**Goal:** Get customers to a quote in **under 2 minutes** with **10-15 questions max** in the primary flow.

**Approach:** 
- Start broad, only drill down when needed
- Use smart defaults and skip logic
- Group related questions visually
- Progressive disclosure (only ask what's necessary)
